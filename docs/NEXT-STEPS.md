# Next Steps (Quick Start)

1) Unzip this pack into your new project root (or copy the `.agent/` folder into the workspace).
2) Open the workspace in Antigravity.
3) In the Agent chat, paste `docs/AG-FIRST-MESSAGE.txt`.
4) Trigger `/bootstrap`.
5) After it runs and the app boots, trigger `/vault`.

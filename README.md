# World-Seed — Antigravity Starter Pack

This pack contains:
- `.agent/rules/` — workspace rules (architecture/security/AI contract)
- `.agent/workflows/` — slash-command workflows
- `schemas/changeset-0.1.schema.json` — ChangeSet v0.1 JSON Schema
- `examples/changeset.example.json` — example ChangeSet
- `docs/` — first message to agent + quick start notes

Copy into your Antigravity workspace, then use the workflows to scaffold and iterate.
